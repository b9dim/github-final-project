# Simple Interest Calculator

A minimal Bash-based simple interest calculator.

## Project Name

**Simple Interest Calculator**

## Overview

This project provides a command-line script that calculates simple interest based on user input.

**Formula:** `Simple Interest = (principal * rate * time) / 100`

## Usage

```bash
bash simple-interest.sh
```

You will be prompted for:
- Principal amount
- Rate of interest (percent per year)
- Time period (years)

## Example

Principal: `1000`
Rate: `5`
Time: `2`

Result: `Simple Interest: 100.00`
